Search Bar Usage

comparasons:
>
<
=
!=
>=
<=
!=> - does not contain
=> - contains

usage:
[...[task-obj-property] [comparason] [value];] (;sort=[sorting id])? (;show-hidden=[bool])? quote values that require spaces

examples:
all tasks with "x" in name, sorted alphabetically: "name !=> x;sort=0"

sorting:
-1 - no sorting, tasks remain in the same order they were recieved in
0 - alphabetical
1 - reverse alphabetical
2 - priority high to low
3 - priority low to high

show hidden:
when set to true shows tasks that don't meet search criteria under those that do (these tasks are sorted using the sort parameter if given)
